Rendezvous

Check Out Github View Live Demo

Rendezvous is a single-page clone of Meetup, a social platform web application where people can connect based on their hobbies or interests.

Rendezvous was designed utilizing PostgreSQL database with Ruby on Rails backend and React / Redux for the frontend.

Users can create an account that are unique and authenticated upon sign-up/login. They have the option to search for currently existing groups based on keyword search and location parameters, and the ability to leave or join them. They can also create their own group, which will give you moderator access to edit or delete them.

Congregate

Check Out Github View Live Demo

Congregate is a data visualization of NYC’s MTA turnstile data set, which tracks the amount of people entering and exiting their stations on a weekly basis. This app’s purpose is to provide the general public and small business owners with an accessible visualization of where and when people congregate to better target their business. Uber drivers, food truck owners, and organizations looking to advertise could benefit from this endeavor.

Congregate is a Javascript project utilizing Google maps and heatmap API, HTML and CSS to render a seamless and aesthetic platform for potential users.

This is still a work in progress as of October 2019. Some updates to be expected in the coming weeks include autoplay feature, addition of entirety of September data, and stylistic improvements.
